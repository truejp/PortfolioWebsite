Garden Dream

Home / Garden Dream

Introducing Garden Dream, a project born out of my passion for gardening and technology. This DIY Greenhouse project aims to simplify the process of growing plants, herbs, and flowers by providing an automated and connected solution. Garden Dream features custom-designed circuit boards and hardware that allow users to remotely monitor and control their greenhouse's climate control units. By setting pre-defined programs, users can ensure that their plants receive optimal growing conditions at all times, resulting in healthy and bountiful harvests. In this project presentation, I will share my journey in creating Garden Dream and the various technical aspects of the project.

The hardware for Garden Dream is built on ATMega Micro Processors and EPS WiFi Chips. These chips allow the control units to receive OTA (over-the-air) updates and connect to the cloud-based backend for remote management. The climate modules are automatically identified and assimilated by the head unit using auto-discovery on the I²C Bus, making it easy for users to expand their setup as needed. Additionally, each user is able to manage their connected devices over the cloud, giving them complete control and visibility of their garden from anywhere.

In the backend of Garden Dream, each user has the ability to install custom plug-ins that give them access to different types of plants. These plug-ins enable the control units to monitor and adjust factors like regular lighting, UV lighting, moisture, and humidity automatically. The frontend is designed to be headless, so that the backend can be very slim and efficient. This allows for a seamless user experience, with easy navigation and minimal lag time between user input and device response. Overall, Garden Dream offers a state-of-the-art solution for those interested in growing their own plants in a controlled, technology-driven environment.
